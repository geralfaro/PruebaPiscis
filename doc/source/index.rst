Welcome to piscis's documentation!
==================================


.. toctree::
   :maxdepth: 2
   :caption: Contents:


Piscis is an app that allows users to vote for the type of interaction observed in images of galaxies pair. 

This is app was built used Django framework. Django is a Python-based free and open-source web framework, 
which follows the model-template-view (MTV) architectural pattern. It is maintained by the Django Software 
Foundation (DSF), an independent organization established as a non-profit.
